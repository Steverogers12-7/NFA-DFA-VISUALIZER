NFA → DFA (Subset Construction)

This tool converts a Non-Deterministic Finite Automaton (NFA) into an equivalent Deterministic Finite Automaton (DFA) using the subset construction algorithm. It also shows step-by-step conversion, transition table, and graphical representation of both automata.

NFA:- NFA (Non-Deterministic Finite Automaton) is a type of finite state machine used in Theory of Computation to recognize patterns and languages.

DFA:-A DFA is a machine in which for every state and input symbol, there is exactly one next state.

All states present in the NFA (comma separated)
Example: q0, q1, q2

Set of input symbols used in the automaton (comma separated)
Example: 0, 1

Initial state where the automaton starts computation
Example: q0

Accepting states of the NFA (comma separated)
Example: q1, q2

Format: Present State,input=Next States
Example: q1,1=q1,q2
